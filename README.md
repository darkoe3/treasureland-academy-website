# Treasureland Academy Website

Modern school website for Treasureland Academy, built with Next.js App Router, JavaScript, Tailwind CSS, Lucide React and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Content and Assets

Place the official school logo at:

```text
public/images/treasureland-logo.jpeg
```

Replace school photographs in `public/images/` using these paths:

